# Task 6 - Sales Trend Analysis Using Aggregations

## Objective
Analyze monthly revenue and order volume using SQL aggregation functions.

## Tools Used
- MySQL
- SQL

## Dataset
online_sales table containing:
- order_id
- order_date
- amount
- product_id

## SQL Concepts Used
- SUM()
- COUNT(DISTINCT)
- GROUP BY
- ORDER BY
- YEAR()
- MONTH()

## Analysis Performed
1. Calculated monthly revenue.
2. Calculated monthly order volume.
3. Grouped sales data by year and month.
4. Sorted results chronologically.

## Outcome
Learned how to analyze sales trends over time using SQL aggregation functions and date-based grouping.
